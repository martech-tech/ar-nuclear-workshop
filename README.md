# ⚛️ โรงไฟฟ้านิวเคลียร์ AR — Workshop วิทยาศาสตร์

WebAR เพื่อการศึกษาสำหรับน้องๆ (ป.4–ม.3) — สแกน QR ด้วยมือถือ → เปิดกล้อง → เล็งมาร์กเกอร์บนกระดาษ → **โรงไฟฟ้านิวเคลียร์ 3 มิติโผล่ขึ้นมา เดินดูรอบๆ ได้จริง** ไม่ต้องติดตั้งแอป พร้อมโหมดเรียนรู้ทีละขั้น ควิซ ใบงาน และคู่มือผู้จัด workshop ครบชุด

## ฟีเจอร์

- 👆 **แตะสำรวจ 10 จุด** — อาคารใหญ่มีป้ายชื่อ จุดรองมีหมุดสีเหลืองกะพริบชวนแตะ กล้องซูมบินเข้าไปดูใกล้ๆ (โหมด 3D) พร้อมการ์ดความรู้ + สเปคจริง + ภาพถ่ายจริง + เกร็ด "รู้หรือไม่?" ทุกจุด
- 🤏 **บีบนิ้วย่อ/ขยาย** — โหมด AR บีบนิ้วปรับขนาดโมเดลบนกระดาษได้เลย (โหมด 3D บีบเพื่อซูมกล้อง)
- 📖 **โหมดเรียนรู้ 9 ขั้น** — เชื้อเพลิงยูเรเนียม → ฟิชชันลูกโซ่ → แท่งควบคุม → น้ำ 2 วงจร → กังหัน 3,000 รอบ → หล่อเย็น → สายส่ง 500,000V → ความปลอดภัย 3 ชั้น พร้อมลูกศรชี้อาคารจริงบนโมเดล
- 🔍 **ส่องข้างใน** — โดมโปร่งใส เห็นเตาปฏิกรณ์เรืองแสง + แท่งควบคุมขยับขึ้นลง
- ⚡ **เม็ดพลังงานวิ่ง** — เห็นเส้นทางพลังงานจากเตาปฏิกรณ์ถึงสายส่งแรงสูง
- ❓ **ควิซ 8 ข้อ** พร้อมคะแนนดาว + คำอธิบายทุกข้อ
- 🏷️ **โลโก้ JKnowledge + Thailand University TCAS Expo** ชัดเจนทุกหน้า (จอ AR, จอ 3D, แผ่นพิมพ์, คู่มือ)
- 🖥️ **โหมด 3D สำรอง** (viewer.html) — ใช้เมื่อไม่มีมาร์กเกอร์/กล้อง หรือฉายโปรเจคเตอร์หน้าห้อง
- 🖨️ **ชุดพิมพ์** — แผ่นมาร์กเกอร์+QR และใบงานนักสำรวจ (print.html) + คู่มือผู้จัดกิจกรรม (workshop.html)

## ไฟล์ในโปรเจกต์

| ไฟล์ | หน้าที่ |
|---|---|
| `index.html` | หน้า AR หลัก (กล้อง + มาร์กเกอร์ Hiro) |
| `viewer.html` | โหมด 3D หมุนดูรอบทิศ (ไม่ใช้กล้อง) — ฟีเจอร์เรียนรู้/ควิซครบเหมือนกัน |
| `print.html` | พิมพ์ 2 หน้า A4: แผ่นมาร์กเกอร์+QR / ใบงานนักสำรวจ |
| `workshop.html` | คู่มือผู้จัด workshop: ผังกิจกรรม 45 นาที วัตถุประสงค์ เฉลย แก้ปัญหา |
| `js/plant-model.js` | โมเดลโรงไฟฟ้า 3D (ใช้ร่วมกันทั้ง AR และ 3D) |
| `js/components.js` | custom components: หอหล่อเย็น LatheGeometry, สายไฟโค้ง, เม็ดพลังงาน, ป้ายไทย, กล้องออร์บิท |
| `js/ui.js` | โหมดเรียนรู้ + ควิซ + ปุ่มควบคุม (ใช้ร่วม 2 หน้า) |
| `libs/` | A-Frame 1.3.0, AR.js 3.4.5, QRCode.js (vendored — ไม่ง้อ CDN ตอนออกบูธ) |

## รันทดสอบในเครื่อง

```bash
npx serve -l 3000 .
# เปิด http://localhost:3000/viewer.html (โหมด 3D ทดสอบได้เลย)
# หน้า AR (index.html) ต้องใช้กล้อง — ทดสอบจริงบนมือถือหลัง deploy
```

## Deploy ขึ้น Vercel

1. push โปรเจกต์นี้ขึ้น GitHub
2. ที่ [vercel.com](https://vercel.com) → **Add New Project** → เลือก repo นี้ → Framework Preset: **Other** → Deploy (ไม่ต้องตั้งค่าอะไรเพิ่ม — เป็น static site ล้วน)
3. ได้ลิงก์ `https://<project>.vercel.app` — เปิด `/print.html` จากลิงก์นั้นเพื่อพิมพ์แผ่นมาร์กเกอร์ (QR จะชี้ URL ที่ถูกต้องให้อัตโนมัติ)

> ⚠️ กล้องทำงานเฉพาะ `https://` — เปิดไฟล์ตรงๆ หรือ `http://` LAN จะขอสิทธิ์กล้องไม่ได้

## วิธีใช้ที่บูธ / ในห้องเรียน

อ่าน **workshop.html** — มีผังกิจกรรม 45 นาที, อุปกรณ์, เฉลยใบงาน, และวิธีแก้ปัญหาเฉพาะหน้าครบ

สรุปสั้น: พิมพ์แผ่นจาก print.html → วางกระดาษบนโต๊ะ → น้องสแกน QR → กด "เริ่มสำรวจ" → เล็งมาร์กเกอร์ → เดินสำรวจ ทำใบงาน แล้วท้าควิซ

## แก้ปัญหาที่พบบ่อย

- **กล้องไม่เปิด** → ต้อง `https://` + กดอนุญาตกล้อง · iPhone ใช้ Safari, Android ใช้ Chrome
- **โมเดลไม่โผล่** → เห็นมาร์กเกอร์เต็มรูปพร้อมขอบขาว แสงพอ กระดาษเรียบ
- **โมเดลสั่น** → ถือนิ่งขึ้น ถอยห่างเล็กน้อย
- **ไม่มีมาร์กเกอร์/กล้องเสีย** → ใช้ `viewer.html` แทนได้ทันที
